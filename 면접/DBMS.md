**1. 인덱스란? b-tree 인덱스란?**

인덱스(INDEX) : 데이터베이스 객체 중 하나로, 테이블에 있는 데이터를 빨리 찾기 위한 용도의 객체이다.

자동 인덱스 : PRIMARY KEY 또는 UNIQUE 제약조건에 의해 자동적으로 생성되는 인덱스 ( 인덱스명은 제약조건명과 동일하게 만들어진다. )
수동 인덱스 : CREATE INDEX 명령문을 실행해서 만드는 인덱스
