### 9-1. JSP 태그의 개념 이해
Servlet은 Java언어를 이용하여 문서를 작성하고, 출력객체를 이용하여 HTML 코드를 삽입하였다.
JSP는 Servlet과 반대로 HTML 코드에 Java언어를 삽입하여 동적 문서를 만들 수 있다.
HTML 코드 안에 Java 코드를 삽입하기 위해서는 태그를 이용해야 한다.

JSP 태그의 종류
- 지시자 : <%@	...	> : 페이지 속성
- 주석 : <%--	...	>
